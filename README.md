# colour picker app

Este é um projeto basico onde eu uso react native com expo, para criar um seletor de cores.

## comandos para instalar o projeto

você vai precisar rodar os seguintes comandos

``` bash
npm install
```


## comandos para iniciar o projeto

você vai precisar rodar os seguintes comandos

``` bash
npx expo start
```
